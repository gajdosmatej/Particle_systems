# Particle_systems

Projekt v rámci stáže Otevřená věda v roce 2021. Projekt se věnoval zkoumání chování tzv. Interacting Particle Systems, což jsou Markovovské řetězce definované na mřížce s lokální přechodovou funkcí pro každý uzel. Tato implementace poskytuje API pro vykreslování animací vývoje takových systémů. Důraz byl kladen hlavně na tzv. Metastable Ising Model a na studium globálně periodického chování pro mřížky ve třech dimenzích. Tři dimenze jsou totiž jediný počet, o němž se neví, jestli umožňuje takové globální chování.
